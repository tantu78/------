# 校园约球平台启动说明

## 部署方式：WAR 包部署到 Tomcat

### 方式一：打包后部署到 Tomcat（推荐）

#### 1. 打包 WAR 文件
```bash
cd backend
mvn clean package
```
打包完成后，WAR 文件位于 `backend/target/campus-ball.war`

#### 2. 部署到 Tomcat
1. 将 `campus-ball.war` 复制到 Tomcat 的 `webapps` 目录下
2. 启动 Tomcat（或等待自动解压）
3. 应用将部署到 `/campus-ball` 路径下

#### 3. 访问地址
- 前端首页：http://localhost:8080/campus-ball/
- API 接口文档：http://localhost:8080/campus-ball/swagger-ui.html

### 方式二：使用嵌入式 Tomcat 开发调试

#### 1. 修改配置
如果需要直接在 IDE 中运行（不使用 WAR 部署），需要：
1. 移除 pom.xml 中 `<packaging>war</packaging>` 改为 `<packaging>jar</packaging>`
2. 恢复 CampusBallApplication.java 为原版（移除继承 SpringBootServletInitializer）
3. 移除 application.yml 中的 `server.servlet.context-path`

#### 2. 运行
```bash
cd backend
mvn spring-boot:run
```
